Tags y Atributos

En HTML, las etiquetas y los atributos son las dos cosas más importantes para construir una página web.

🔖 ¿Qué son las etiquetas (tags)?

Las etiquetas son palabras que van entre < > y sirven para definir qué es cada elemento en la página.

Le dicen al navegador:

  • Esto es un título
  • Esto es un párrafo
  • Esto es una imagen
  • Esto es un enlace
  • Esto es un botón

✅ Ejemplo de etiquetas:

      
        <h1>Esto es un título</h1>
        <p>Esto es un párrafo</p>
        <a href="#">Esto es un enlace</a>

Casi todas las etiquetas tienen:

  • 🔓 Etiqueta de apertura
  • 🔒 Etiqueta de cierre
  • Todo lo que este adentro es el contenido

🔩 ¿Qué son los atributos?

Los atributos dan información extra sobre una etiqueta.

Van dento de la etiqueta de apertura

Sirven para:

  • Indicar la dirección de una imagen
  • Decir a dónde va un enlance
  • Cambiar tamaño
  • Agregar colores
  • Identificar elementos
  • Etc.

📌 Ejemplo con enlace:

      
        <a href="https://www.ejemplo.com">Esto es un enlace</a>

👉 El atributo href le dice al enlace a qué página ir.

👉 Las etiquetas crean el elemento

👉 Los atributos lo modifican

Insensibilidad a mayúsculas

La case insensitivity (insensibilidad a mayúsculas y minúsculas) en HTML es algo muy importante cuando estás empezando.

🔤 ¿Qué significa que HTML sea case insensitive?

Significa que HTML NO distingue entre mayúscla y minúsculas:

✅ Las etiquetas

✅ Los atributos

El navegador entiende lo mismo aunque escribas:

      
        <h1>Esto es un título</h1>
        <H1>Esto es un título</H1>
        <h1>Esto es un título</H1>

⚠️ PERO OJO: Los valores de los atributos SÍ pueden ser sensibles

Esto NO siempre es lo mismo:

      
        <a href="https://www.ejemplo.com">Esto es un enlace</a>
        <a href="HTTPS://WWW.EJEMPLO.COM">Esto es un enlace</a>

🔎 ¿Qué son las Entidades HTML (HTML Entities)?

Las entidades HTML son códigos especiales que se usan para mostrar caracteres o símbolos en una página web.

Son necesaruis cyabdi quieres escribir:

  • Símbolos especiales (& < >)
  • Acentos (á, é, í, ó, ú)
  • Expacios especiales ( )
  • O caracteres que HTML usa como código (< >)

🧩 ¿Cómo se escribe una entidad HTML?

Tadas tienen esta estructura: &nombre;

Siempre:

  • Empiezan con &
  • Terminan con ;

✏️ Ejemplos importantes:

Símbolo Entidad HTML
& &amp;
< &lt;
> &gt;
á &aacute;
é &eacute;
í &iacute;
ó &oacute;
ú &uacute;

¿Qué son los comentarios en HTML?

Los comentarios son text que escribes destro del código pero NO se muestran en la página web.

Sirven para:

  • Explicar partes del código
  • Dejar notas para ti o para otros desarrolladores
  • Desactivar temporalmente código sin borrarlo
  • Trabajar en equipo

¿Cómo se escribe un comentario?

      
        <!-- Este es un comentario -->

Comentarios de varias líneas:

      
        <!--
          Este es un comentario
          de varias líneas
        -->
