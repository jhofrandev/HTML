Lenguaje de Marcado

Un lenguaje de marcado es una forma de escribir texto donde usas etiquetas para decirle a la computadora qué es cada parte del contenido.

❌ No le dices cómo se ve. ✅ Le dices qué significa.

Ejemplo en la vida real:

Imagina que tienes este texto:

Hola, mi nombre es Juan

Para una persona eso está claro, pero para una computadora eso es solo texto plano.

Con un lenguaje de marcado, tú puedes explicarle

<p>Hola, mi nombre es Juan</p>

Ahora la computadora entiende: ✅ Esto es un párrafo

¿Por qué se llama "Marcado"?

Porque estás marcando el contenido con etiquetas.

Como si usaras un resaltador para decir:

  • Esto es un título
  • Esto es un párrafo
  • Esto es un enlace
  • Esto es una imagen

Pero en vez de colores, usas etiquetas como: <h1>, <p>, <a> <img>
