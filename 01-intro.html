HTML

HTML significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto).

Es el lenguaje básico (lenguaje de marcado) que se usa para crear páginas web.

Todo lo que ves en un sitio web - textos, imágenes, enlaces, tablas - está organizado usando HTML

¿Para qué sirve HTML?

  • ✅ Estructurar el contenido de una página
  • ✅ Decir qué es un título
  • ✅ Qué es un párrafo
  • ✅ Dónde va una imagen
  • ✅ Crear enlaces (link)
  • ✅ Botones, formularios, listas, etc.

Si se fijan HTML funciona con etiquetas (tags), que casi siempre vienen en pares
