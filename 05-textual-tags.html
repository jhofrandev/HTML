Encabezados HTLM: <h1> hasta <h6>

Estas etiquetas se llaman encabezados (headings) y sirven para crear títulos y subtilos dentro de tu pagina web.

Su funcion prioncipal es:

  • ✅ Organizar el contenido
  • ✅ Crear estructura
  • ✅ Definir jerarquia
  • ✅ Facilitar la lectura
  • ✅ Ayudar a Google a entender tu pagina

Cuantos hay?

👉 6 niveles de encabezados

Etiqueta Nivel Uso
<h1> Mas importante Titulo principal
<h2> Alto Secciones
<h3> Medio Subsecciones
<h4> Bajo Detalles
<h5> Menor Sub-detalles
<h6> Menos importante Informacion minima

Como funcionan?

Los encabezados funcionan como el indice de un libro

Por ejemplo:

      
        <h1> = Titulo del libro
        <h2> = Capitulos
        <h3> = Subcapitulos
        <h4> = Temas
        <h5> = Subtemas
        <h6> = Notas

Con respecto a su jerarquia es importante no saltarse niveles; empieza con <h1> y ve bajando. Solo usar un <h1> por página.

Etiqueta <title>

La etiqueta <title> se utiliza para definir el título de una página web. Este título se muestra en la pestaña del navegador y en los resultados de los motores de búsqueda.

Ejemplo:

      
        <title>Mi Página de Ejemplo</title>

Caracteristicas:

  • ✅ Se coloca dentro del <head>
  • ✅ Solo puede haber uno por página
  • ✅ Debe ser corto y descriptivo
  • ✅ Ayuda al SEO
  • ✅ Se muestra en la pestaña del navegador

Etiqueta <p>

La etiqueta <p> se utiliza para definir un párrafo de texto. Es una de las etiquetas más utilizadas en HTML.

Ejemplo:

      
        <p>Este es un párrafo de ejemplo.</p>

Tener en cuenta que capa parrafo empieza una nueva linea, ocupa todo el ancho disponible y se separa de otros elementos con un espacio en blanco (margin).

Etiqueta <hr>

La etiqueta <hr> se utiliza para definir una línea horizontal que separa el contenido de una página web. Es una etiqueta que no tiene cierre y se utiliza para separar el contenido de una página web.

Ejemplo:

      
        <hr>

Caracteristicas:

  • ✅ Es una etiqueta vacia
  • ✅ No tiene contenido
  • ✅ No tiene cierre
  • ✅ Es autocontenida

<br> (Line Break)

Sirve para crear un salto de linea dentro del texto.

Ejemplo:

      
        <p>Hola<br>Mundo</p>

Resultado:

Hola
Mundo

🅱️ <b> vs 💪 <strong>

Aunque ambas muestran texto en negrita, la etiqueta <b> se utiliza para resaltar texto visualmente sin transmitir importancia adicional. Mientras que la etiqueta <strong> define texto con fuerte importancia o enfasis dentro del documento.

Recordar que en HTML existe algo llamado semantica, que quiere dcir esto, pues que el contenido tiene un significado o deberia de tenerlo y no es solo contenido visual y la etiqueta strong tiene importancia semantica mientra que b no.

Ejemplo:

      
        <p>El <b>Chocolate</b> es delicioso.</p>
        <p><strong>No olvides guardar tu trabajo.</strong></p>

Es importante tener claro el tema de la semantica de las etiquetas ya que esto ayuda a Google a entender mejor tu contenido, igual que a los lectores de pantalla y sin duda mejora el SEO y accesibilidad de la pagina

🧾 La etiqueta <pre>

El elemento <pre> representa texto que debe mostrarse exactamente como esta escrito en el archivo HTML.

Que significa "preformateado"

❌ HTML elimina los espacios extras

❌ HTML elimina los saltos de linea

✔️ <pre> conserva los espacios

✔️ <pre> conserva las tabulaciones

✔️ <pre> conserva los saltos de linea

✔️ <pre> conserva las sangrias

✔️ <pre> conserva el formato original

Ejemplo:

      
        <pre>
          function saludar() {
            console.log('Hola mundo')
          }
        </pre>

💻 Para que se usa?

  • ✅ Mostrar codigo
  • ✅ Mostrar poemas
  • ✅ Mostrar direcciones
  • ✅ ASCII Art
  • ✅ Texto estructurado
  • ✅ Tablas de texto

✍️ <i> vs 🗣️ <em>

Tanto <i> como <em> muestran texto en cursiva, pero tienen propositos diferentes.

<i> se utiliza para mostrar texto en cursiva sin añadir enfasis semantica, por ejemplo:

  • Esta en otro idioma
  • Es un pensamiento
  • Es un termino tecnico
  • Es un nombre cientifico
  • tiene otra voz o tono

<em> marca texto que tiene enfasis dentro del contenido y est permite que:

  • Le da importancia
  • Cambiar el significado de la oracion
  • Mejorar la accesibilidad
  • Mejorar comprension
  • Mejorar SEO indirectamente

🟨 <mark>

La etiqueta <mark> se utiliza para definir texto resaltado o marcado dentro de un documento.

Se puede usar para:

  • ✅ Destacar indonrmacion importante
  • ✅ Resaltar palabras clave
  • ✅ Marcar resultados de busqueda
  • ✅ Llamar la atencion del lector
  • ✅ Notas importantes
  • ✅ Mensajes destacados

Marcar la diferencia con b y strong. mientra que b es solo negrita y strong es marca importancia, mark es mas resaltado visual.

Ejemplo:

      
        <p>El <mark>Chocolate</mark> es delicioso.</p>

resultado:

El Chocolate es delicioso.

🔽 <sub> y 🔼 <sup>

Vamos con dos etiquetas muy usadas en matematicas, quimica y notas

🔼 Etiqueta <sup> (Superescrito)

El elemento <sup> muestra texto mas alto que la linea normal, generamente en tamaño mas pequeño.

Ejemplo:

      
        <p>x<sup>2</sup></p>

Resultado:

x2

Donde se usa?

  • Potencias matematicas -> x2
  • Notas al pie -> texto1
  • Numeross ordinales -> 1o, 2o, 3o
  • Unidades de medida -> m2, km2

🔼 Etiqueta <sub> (Subescrito)

El elemento <sub> muestra texto mas bajo que la linea normal, generamente en tamaño mas pequeño.

Ejemplo:

      
        <p>H<sub>2</sub>O</p>

Resultado:

H2O

Donde se usa?

  • Formulas quimicas -> CO2
  • Variables matematicas -> x1
  • Indices -> A1
  • Notaciones tecnicas

🔤 <small>

La etiqueta <small> se utiliza para definir texto mas pequeño que el texto normal.

Se puede usar para:

  • ✅ Texto legal
  • ✅ Notas al pie
  • ✅ Copyright
  • ✅ Avisos legales
  • ✅ Texto pequeño

Ejemplo:

      
        <p>El <small>Chocolate</small> es delicioso.</p>

resultado:

El Chocolate es delicioso.

🔗 Links (enlaces) en HTML

La etiqueta <a> (ancla) crea un enlace que permite ir de una pagina a otra.

Que puede hacer un link?

Un enlace puede llevarte a:

  • ✅ Otra página web
  • ✅ Otra página del mismo sitio
  • ✅ Una sección de la misma página
  • ✅ Llamar a un numero de telefono
  • ✅ Enviar un correo electrónico
  • ✅ Descargar un archivo
  • ✅ Abrir una imagen
  • ✅ Abrir un video
  • ✅ Abrir un documento
  • ✅ Abrir una aplicacion

Sintaxis basica

      
        <a href="https://www.google.com">Ir a Google</a>

Tipos de enlaces

Existen varios tipos de enlaces:

  • 🌍 1. Enlace externo
    •         
          <a href="https://www.youtube.com">Youtube</a>
  • 📄 2. Enlace interno
    •         
          <a href="contacto.html">Contacto</a>
  • 📍 3. Enlace dentro de la misma página
    •         
          <a href="#info">Ver mas informacion</a>
          ...
          <h2 id="info">Informacion</h2>
  • 📧 4. Enviar correo:
    •         
          <a href="mailto:correo@email.com">Enviar correo</a>
  • 📱 5. Llamar por teléfono:
    •         
          <a href="tel:+573001234567">Llamar ahora</a>

⚙️ Atributos importantes

🪟 Abrir en nueva pestaña:

      
        <a href="https://www.google.com" target="_blank">Ir a Google</a>

💬 Texto emergente:

      
        <a href="https://www.google.com" title="Ir a Google">Ir a Google</a>

Descargar archivo:

      
        <a href="archivo.pdf" download>Descargar archivo</a>

🖼️ Imagen como enlace

      
        <a href="https://www.google.com">
          <img src="imagen.jpg" alt="Imagen">
        </a>

🔤 <del> y <ins>

La etiqueta <del> se utiliza para definir texto que ha sido eliminado del documento.

La etiqueta <ins> se utiliza para definir texto que ha sido insertado en el documento.

Ejemplo:

      
        <p>Precio: <del>$10</del> $5</p>

Resultado:

Precio: $10 $5

Ejemplo:

      
        <p>Ahora cuesta <ins>$10</ins></p>

Resultado:

Ahora cuesta $10

🔤 <u>

La etiqueta <u> se utiliza para definir texto que ha sido subrayado.

Ejemplo:

      
        <p>El <u>Chocolate</u> es delicioso.</p>

Resultado:

El Chocolate es delicioso.

Nota: Se recomienda usar <ins> en lugar de <u> para texto subrayado.
