📊 Tablas en HTML

El elemento tabla <table> representa datos tabulares organizados en filas y columnas, las tablas se usan para:

  • Horarios
  • Estadisticas
  • Listas de productos
  • Calificaciones
  • Reportes

🧠 Como funciona una tabla?

Una tabla es un conjunto estructurado de datos formado por filas y columnas, donde las intersecciones crean celdas

Estructura basica de una tabla

Los elementos principales de una tabla son:

  • <table>: Define la tabla
  • <tr>: Define una fila
  • <td>: Define una celda
  • <th>: Define un encabezado

Ejemplo de tabla basica

Nombre Edad Ciudad
Juan 25 Lima
Maria 22 Bogota

Estructura semantica avanzada

La estructura semantica de una tabla permite organizar mejor los datos y mejorar la accesibilidad

  • <thead>: Define el encabezado de la tabla
  • <tbody>: Define el cuerpo de la tabla
  • <tfoot>: Define el pie de la tabla

Ejemplo de tabla semantica

Nombre Edad Ciudad
Juan 25 Lima
Maria 22 Bogota
Total 2 2

Combinacion de celdas

Las celdas se pueden combinar usando los atributos:

  • colspan: Combina celdas horizontalmente
  • rowspan: Combina celdas verticalmente

Ejemplos:

Total
Juan 25
Maria 22
Total
Juan 25
Maria 22
