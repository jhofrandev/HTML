📦 1. Etiqueta <div>

El elemento <div> es un contenedor genérico que se usa para agrupar contenido.

Se usa para agrupar contenido y facilitar el diseño usando atributos como class o id.

Ejemplo:

      
        <div class="contenedor">
          <p>Párrafo 1</p>
          <p>Párrafo 2</p>
        </div>

📦 2. Etiqueta <span>

El elemento <span> es un contenedor en línea usado para marcar partes del texto.

  • ✅ Agrupa pequeñas porciones de texto
  • ✅ Se usa para aplicar estilos a palabras o frases
  • ✅ Es un elemento en línea

Ejemplo:

      
        <p>Este es un texto con una palabra en <span class="resaltado">negrita</span>.</p>
