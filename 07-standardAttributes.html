🌍 Que son los Global Attributes?

Los atributos globales pueden utilizarse en cualquier elemento HTML para mejorar comportamiento, estilo o accesibilidad.

Sintaxis de un atributo

      
        <elemento atributo="valor">Contenido</elemento>

Nota: Los atributos siempre se escriben en minúsculas y los valores deben ir entre comillas dobles o simples.

🔑 Atributos Globales mas importantes

🆔 1. id

El atributo id define un identificador unico dentro del documento.

      
        <div id="mi-id">Contenido</div>

🧱 2. class

Class asigna uno o mas nombres clase para aplicar estilos o manipulacion con javascript 

      
        <div class="clase1 clase2">Contenido</div>

🎨 3. style

Style permite aplicar estilos CSS en linea.

      
        <div style="color: blue;">Contenido</div>

🧠 4. data-* (Custom Data Attributes)

data-* permite almacenar datos personalizados dentro de HTML.

      
        <div data-id="123" data-nombre="Juan">Contenido</div>

💬 5. title

Title proporciona informacion adicional sobre un elemento. Se muestra como tooltip cuando el usuario pasa el mouse sobre el elemento. 

      
        <img src="imagen.jpg" alt="Descripcion" title="Titulo de la imagen">

🌐 6. lang

Lang especifica el idioma del contenido del elemento. Es importante para la accesibilidad y SEO. 

      
        <p lang="es">Este texto esta en español.</p>

↔️ 7. dir

Dir especifica la direccion del texto. Puede ser "ltr" (izquierda a derecha) o "rtl" (derecha a izquierda). 

      
        <p dir="rtl">Este texto esta en arabe.</p>

👁️ 8. hidden

Hidden indica que el elemento no es visible para el usuario. Se usa para ocultar elementos condicionalmente. 

      
        <div hidden>Este elemento esta oculto.</div>

🖱️ 9. draggable

Draggable indica si un elemento puede ser arrastrado. Puede ser "true" o "false". 

      
        <div draggable="true">Este elemento puede ser arrastrado.</div>

🧑‍💻 10. contenteditable

Contenteditable indica si el contenido de un elemento puede ser editado por el usuario. Puede ser "true" o "false". 

      
        <div contenteditable="true">Este elemento puede ser editado.</div>

📇 11. tabindex

Tabindex indica el orden en que los elementos deben ser enfocados cuando el usuario presiona la tecla Tab. Puede ser un numero entero positivo o negativo. 

      
        <div tabindex="1">Este elemento sera enfocado primero.</div>
