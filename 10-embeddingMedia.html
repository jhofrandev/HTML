El elemento <img>

El elemento <img> se usa para incrustar imagenes en documentos HTML

Sisntaxis minima correcta

      
        <img src="ruta/a/la/imagen.jpg" alt="Descripción de la imagen">

El atributo <alt> proporciona texto alternativos cuando la imagen no puede mostrarse y es clave para accesibilidad.

Mejores paracticas para <img>

  • Siempre incluye el atributo <alt> con una descripción concisa de la imagen.
  • Usa rutas relativas para las imágenes cuando sea posible. Esto hace que el sitio sea más fácil de mantener.
  • Usa atributos width y height para especificar las dimensiones de la imagen. Esto ayuda a prevenir cambios en el diseño mientras la imagen se carga.

Priority Hints en imagenes

Priority Hints (Fetch Priority) permiten indicar la prioridad relativa de descarga de un recurso, sugiriendo al navegador sobre que recursos carga antes o despues.

loading: cuando cargar

Valores:

Valor Significado
lazy La imagen se carga cuando se necesita
eager La imagen se carga inmediatamente
(default) El navegador decide cuando cargar la imagen 
      
        <img src="imagen.jpg" alt="Descripción" loading="lazy">

El atributo <loading> controla el comportamiento de carga (lazy/eager)

fetchpriority: prioridad de descarga

Valores:

Valor Significado
high Indica que el recurso es prioritario
low Indica que el recurso no es prioritario
(default) El navegador decide cuando cargar la imagen 
      
        <img src="imagen.jpg" alt="Descripción" fetchpriority="high">

El atributo <fetchpriority> indica al navegador cómo priorizar la descarga de un recurso.

<img> vs <figure>

El elemento <img> es el medio básico para mostrar una imagen sin contenido adicional

El elemento <figure> representa contenido autocontenido que puede moverse sin afectar el flujo del documento 

      
        <figure>
          <img src="imagen.jpg" alt="Descripción">
          <figcaption>Descripción de la imagen</figcaption>
        </figure>

<figure> se usa para organizar semánticamente imágenes con contexto, mientras <img> es solo el recurso visual.
