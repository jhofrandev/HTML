📋 List & Types en HTML

HTML tiene un conjunto de elementos que permiten definir diferentes tipos de listas: unordered, ordered y description list, lo que te puede servir para:

👉 Organizar informacion

👉 Agrupar elementos relacionados

👉 Mostrar secuencias

👉 Crear menus

👉 Mostrar definiciones

Tipo Elemento Descripcion
Unordered List ul Lista no ordenada
Ordered List ol Lista ordenada
Description List dl Lista de definiciones

🔘 1. Unordered List (Lista Desordenada)

Una lista desordenada comenza con la etiqueta <ul> y cada elemento de la lista se define con la etiqueta <li>

Se usa cuando el orden no importa:

  • Lista de compra
  • Nombres
  • Categorias

Ejemplo:

  • Leche
  • Pan
  • Huevos

Sintaxis

      
        <ul>
          <li>Elemento 1</li>
          <li>Elemento 2</li>
          <li>Elemento 3</li>
        </ul>

🔢 2. Ordered List (Lista Ordenada)

Una lista ordenada comenza con la etiqueta <ol> y cada elemento de la lista se define con la etiqueta <li>

Se usa cuando el orden importa:

  1. Pasos de una receta
  2. Instrucciones
  3. Secuencias

Ejemplo:

  1. Precalentar horno a 180°C
  2. Mezclar ingredientes secos
  3. Agregar ingredientes humedos
  4. Hornear por 30 minutos

Sintaxis

      
        <ol>
          <li>Elemento 1</li>
          <li>Elemento 2</li>
          <li>Elemento 3</li>
        </ol>

Atributos de Ordered List

La etiqueta <ol> tiene los siguientes atributos:

  • type: Define el tipo de numeracion
    • 1: Numeros (default)
    • a: Letras minusculas
    • A: Letras mayusculas
    • i: Romanos minusculas
    • I: Romanos mayusculas
  • start: Define el numero de inicio
    • 1: Empieza en 1 (default)
    • 2: Empieza en 2
    • 3: Empieza en 3

Ejemplos

      
        <ol type="a">
          <li>Elemento 1</li>
          <li>Elemento 2</li>
          <li>Elemento 3</li>
        </ol>

📖 3. Description List (Lista de Definiciones)

Una lista de definiciones comenza con la etiqueta <dl> y cada elemento de la lista se define con la etiqueta <li>

Se usa cuando el orden importa:

Termino 1
Definicion 1
Termino 2
Definicion 2
Termino 3
Definicion 3

Ejemplo:

HTML
HyperText Markup Language
CSS
Cascading Style Sheets
JavaScript
JavaScript

Sintaxis

      
        <dl>
          <dt>Termino 1</dt>
          <dd>Definicion 1</dd>
          <dt>Termino 2</dt>
          <dd>Definicion 2</dd>
          <dt>Termino 3</dt>
          <dd>Definicion 3</dd>
        </dl>

🧬 4. Nested List (Lista Anidada)

Una lista anidada es una lista dentro de otra lista. Se usa cuando se quiere mostrar una jerarquia de elementos.

Ejemplo:

  • Frutas
    • Manzana
    • Banana
    • Naranja
  • Verduras
  • Carnes
