¿Qué es <!DOCTYPE>?

<!DOCTYPE html>

El DOCTYPE (Document Type Declaration) es una declaración que se coloca al inicio de un documento HTML para indicar al navegador qué versión de HTML se está utilizando. En HTML5, la declaración es simplemente <!DOCTYPE html>, lo que indica que el documento sigue las reglas de HTML5. Esta declaración ayuda a los navegadores a renderizar la página correctamente y a evitar problemas de compatibilidad.

⚠️ Importante

No es una etiqueta HTML, es una directiva que contiene meta-información del documento.

Etiqueta <html>

La etiqueta <html> es fundamental porque actúa como el contenedor principal de todo un documento web. Se le conoce como la raíz 🌲 del documento, ya que todos los demás elementos (como el encabezado, el cuerpo, las imágines y los párrafos) deben estar dentro de ella.

Cuando un nevegador 🌐 abre un archivo, la etiqueta <html> le indica: "Todo lo que esta aquí dentro es código HTML". Generalmente, se ve así: 

      
        <!DOCTYPE html>
        <html lang="es">
          <head>
            ...
          </head>
          <body>
            ...
          </body>
        </html>

El atributo lang="es". Este es muy importante porque le dice a los navegadores y a los lectores de pantalla en que idioma está escrita la página, lo cual ayuda mucho a la accesibilidad.

Etiqueta <head>

Es la cabecera de tu página web. Contiene información sobre el documento HTML, pero esa información NO se muestra en la página

Es decir:

El <head> contiene metadatos (información sobre la página) como su título, enlaces a estilos, scripts, etc.

El contenido del <head> no es visible para los usuarios, pero es crucial para el funcionamiento correcto de la página.

¿Qué son las etiquetas meta en HTML?

Son etiquetas HTML que proporcionan información sobre la página web. Esta etiqueta se coloca dentro del <head> y no es visible para los usuarios.

🎯 ¿Para qué sirven?

Las etiquetas <meta> ayudan a:

✅ Mostrar bien el texto (acentos, ñ, etc.)

✅ Hacer la página responsive en el celular

✅ Describir tu página en Google

✅ Decir quién es el autor

✅ Controlar si Google indexa tu página

✅ Configurar idioma o codificación

Ejemplo completo:

      
        <!DOCTYPE html>
        <html lang="es">
          <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <meta name="description" content="Una página de ejemplo para aprender HTML">
            <meta name="author" content="Tu Nombre">
            <title>Mi Página de Ejemplo</title>
          </head>
          <body>
            ...
          </body>
        </html>

¿Qué es la etiqueta <body>?

La etiqueta <body> es el contenedor principal del contenido visible de una página web. Todo lo que quieras mostrar a los usuarios, como texto, imágenes, videos, enlaces, etc., debe estar dentro de esta etiqueta.
