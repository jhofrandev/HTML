¿Qué es <!DOCTYPE>?

<!DOCTYPE html>

El DOCTYPE (Document Type Declaration) es una declaración que se coloca al inicio de un documento HTML para indicar al navegador qué versión de HTML se está utilizando. En HTML5, la declaración es simplemente <!DOCTYPE html>, lo que indica que el documento sigue las reglas de HTML5. Esta declaración ayuda a los navegadores a renderizar la página correctamente y a evitar problemas de compatibilidad.

⚠️ Importante

No es una etiqueta HTML, es una directiva que contiene meta-información del documento.

Etiqueta <html>

La etiqueta <html> es fundamental porque actúa como el contenedor principal de todo un documento web. Se le conoce como la raíz 🌲 del documento, ya que todos los demás elementos (como el encabezado, el cuerpo, las imágines y los párrafos) deben estar dentro de ella.

Cuando un nevegador 🌐 abre un archivo, la etiqueta <html> le indica: "Todo lo que esta aquí dentro es código HTML". Generalmente, se ve así: 

      
        <!DOCTYPE html>
        <html lang="es">
          <head>
            ...
          </head>
          <body>
            ...
          </body>
        </html>

El atributo lang="es". Este es muy importante porque le dice a los navegadores y a los lectores de pantalla en que idioma está escrita la página, lo cual ayuda mucho a la accesibilidad.

Etiqueta <head>

La etiqueta <head> es ek centro de control 🧠 de tu página web. A diferencia del cuerpo, donde colocamos todo lo que el usaurio ve, el <head> contiene información técnica que el navegador y los buscadores necesitan para entender y mostrar tu sitio correctamente.
